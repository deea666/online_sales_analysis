\# Online Sales Analysis



\## Autor

Andreea Voicu



\## Descriere

Acest proiect a fost realizat pentru gestionarea produselor și a coșului de cumpărături folosind Python și GitHub.



\## Clase



\### Product

\- nume produs

\- preț

\- cantitate



\### ProductManager

\- adăugare produse

\- afișare produse

\- eliminare produse

\- calcul valoare inventar



\### Cart

\- adăugare produse în coș

\- afișare produse din coș

\- calcul valoare totală



\## Tehnologii utilizate

\- Python

\- Git

\- GitHub
